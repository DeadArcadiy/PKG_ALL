Лабораторная работа 2
Цели:
Выполнение настоящей работы имеет целью закрепление теоретического
материала и практическое освоение основных возможностей по:
• работе с различными форматами хранения растровых изображений;
• получению информации об изображении, хранящемся в файле.
Задача:
Написать приложение/веб-приложение, считывающее из графического
файла/файлов (должна быть возможность указать папку, содержащую до 100000
файлов) основную информацию об изображении. Обязательно надо отображать
следующее (для удобства результат можно представлять в таблице):
имя файла;
размер изображения (в пикселях);
разрешение (dot/inch);
глубина цвета;
сжатие (для тех типов файлов, в которых оно может быть задано).
Реализация:
Проект написан на языке С++ на фреймворке QT. Были использованы библиотеки QDir,QFileSystemModel,QFileDialog
для работы с файлами.Создан виджет на основе стандартного класса QWidget.Навигация по файловой системе реализована
при помощи listView c QDir и QFileSystemModel вариант выбора одного эллемента. Либо с помощью кнопки "Choose" и
QFileDialog вариант выбора нескольких файлов одновременно. После выбора файлов отроется QTable с данными о файлах.
Использование:
После запуска программы следует открыть нужные файлы любым из перечисленных способов.При выборе при помощи QFileDialog
можно выбрать несколько файлов и получить таблицу. При выборе из listView в цетре можно получить информацию об одном файле.
