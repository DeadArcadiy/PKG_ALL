Лабораторная №3
Основы обработки цифровых изображений
Вариант 3
Задачи:
Написать приложение/веб-приложение, реализующее указанные в варианте
методы обработки изображений. 
Реализация высокочастотных фильтров(увеличение резкости)
Построение и эквализация гистограммы изображения+линейное контрастирование
Реализация:
Программа написана на языке Python с использованием следующих библиотек:
Matplotlib
Numpy
OpenCV
TKinter
PIL
Основа приложения написана на TKinter, для функций обработки изображений используется OpenCV, а для постороения
графиков используется MatPlotlib
Использование программы:
При запуске программа автоматически использует файлы high.jpg и lin.jpg.
Затем отображается результат работы программы.
В первом ряду слева направо отображаются оригинал high.jpg оригинал, лапласиан, лапласиан-гаусян
Во втором ряду показана гистограмма lin.jpg и 2 оригинала этого изображения (для удобства последующего сравнения)
В третьем ряду показывается гистограмма после применения эквализации, полученное изображение при эквализации
и изборажение полученное с помощью линейного контрастирования.
При желании использовать данную программу на других изображениях необходимо закинуть картинки с названиями high.jpg или lin.jpg 
в папку с exe.
